Com a propriedade CSS3 border-image, você pode definir uma imagem para ser usada como a borda em torno de um elemento.

Border-radius

Adicione bordas arredondadas a um elemento

Box-shadow

Adicione uma sombra de caixa a um elemento

Pseudo-elementos:

::After

O seletor :: after após inserir algo após o conteúdo de cada elemento selecionado (s).Use a propriedade de conteúdo para especificar o conteúdo a ser inserido.Use o seletor :: before para inserir algo antes do conteúdo.

::Before

O :: antes do seletor insere algo antes do conteúdo de cada elemento selecionado (s).Use a propriedade de conteúdo para especificar o conteúdo a ser inserido.Use o seletor :: after para inserir algo depois do conteúdo.

::First-letter

O seletor :: first-letter é usado para adicionar um estilo à primeira letra do seletor especificado.

::First-line

O seletor :: first-line é usado para adicionar um estilo à primeira linha do seletor especificado.

::Selection

O seletor de seleção :: corresponde à parte de um elemento que é selecionado por um usuário.Somente algumas propriedades CSS podem ser aplicadas ao seletor de seleção: cor, fundo,cursor e contorno.

Pseudo-classes:

: fixirst-child

O seletor first-child é usado para selecionar o seletor especificado, somente se ele for o primeiro filho de seu pai.

:first-of-type

O seletor first-of-type corresponde a cada elemento que é o primeiro filho, de um tipo específico, de seu pai.

:focus

O seletor de foco: é usado para selecionar o elemento que tem foco.

:hover

O seletor: hover é usado para selecionar elementos quando você mouse sobre eles.

:only-child

O seletor: only-child corresponde a cada elemento que é o único filho de seu pai.

:only-of-type

O seletor de somente-tipo corresponde a cada elemento que é o único filho de seu tipo, de seu pai.

:not()

A negação CSS pseudo-classe , :not(X)é uma notação funcional tomar um simples selector X como um argumento. Ele corresponde a um elemento que não é representado pelo argumento. X não deve conter outro seletor de negação.

: nth-child ()

A: nth-child ( n ) selector corresponde a cada elemento que é o n th criança, independentemente do tipo, de seu pai.N pode ser um número, uma palavra-chave ou uma fórmula.

:active

O seletor ativo é usado para selecionar e estilizar o link ativo.Um link torna-se ativo quando você clica nele.

:checked

O atributo checked é um atributo booleano.Quando presente, especifica que um elemento <input> deve ser pré-selecionado (verificado) quando a página for carregada. O atributo verificado pode ser usado com <input type = "checkbox"> e <input type = "radio"> .O atributo checked também pode ser definido após o carregamento da página, com um JavaScript.

:disabled

O atributo disabled é um atributo booleano.Quando presente, ele especifica que o elemento <input> deve ser desabilitado.Um elemento de entrada desativado é inutilizável e não pode ser clicado.O atributo disabled pode ser configurado para impedir que um usuário use o elemento <input> até que alguma outra condição seja atendida (como selecionar uma caixa de seleção, etc.). Em seguida, um JavaScript pode remover o valor desativado e tornar o elemento <input> utilizável.

:enabled

O seletor: habilitado corresponde a cada elemento habilitado (usado principalmente em elementos de formulário).

:empty

O :empty pseudo-classrepresenta qualquer elemento que não tem filhos. Somente nós de elementos e texto (incluindo espaços em branco) são considerados. Comentários ou instruções de processamento não afetam se um elemento é considerado vazio ou não.

:target

O atributo de destino especifica onde abrir o documento vinculado.

Parte 2

Address

A tag <address< define as informações de contato para o autor / proprietário de um documento ou artigo.

Progress

Tag representa o progresso de uma tarefa.

Map

A tag <map> é usada para definir um mapa de imagem do lado do cliente. Um mapa de imagem é uma imagem com áreas clicáveis.

Area

A tag <area> define uma área dentro de um mapa de imagem (um mapa de imagem é uma imagem com áreas clicáveis).

Mark

O <mark> define o texto marcado.Use o se quiser destacar partes do seu texto.

Embed

A tag <embed> define um contêiner para um aplicativo externo ou conteúdo interativo (um plug-in).

Iframe

A tag <iframe> especifica um quadro embutido.Um quadro embutido é usado para incorporar outro documento dentro do documento HTML atual.

DL

A tag <dl> define uma lista de descrição.

DD

A tag <dd> é usada para descrever um termo / nome em uma lista de descrição.

DT

The <dt> tag defines a term/name in a description list.

Colgroup

A tag <colgroup> especifica um grupo de uma ou mais colunas em uma tabela para formatação.

Blockquote

A tag <blockquote> especifica uma seção que é citada de outra fonte.

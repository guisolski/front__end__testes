Seletores

Os seletores são utilizados para referenciar algo no css do html. Existem inumeors tipos de seletores, por exemplo:

  • Tag: você chama a tag no css, e aplica os estilos as essas tags
  • Id: este você coloca após a tag de abertura, nele você indica uma tag name onde você a referencia no css apos #.
  • Class: este como o id é colocado após a tag de abertura, porem no css ele é chamado após o . (ponto). Ele tem uma vantagem que é permitir chamar outros inumnerados ou tags que estão dentro dele.

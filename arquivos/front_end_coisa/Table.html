Table

O elemento HTML Table <table> representa dados em duas dimensões ou mais.

THEAD

O elemento <THEAD> define um grupo de linhas de cabeçalho em uma tabela. A TABELA pode ter um THEAD , que deve seguir qualquer CAPTION , COL ou COLGROUP elementos, e precedem o opcional TFOOT e exigiu TBODY elementos.

TBODY

A tag TBODY define o corpo de dados de uma tabela.

td

A celula da tabela HTML elemento <td> define uma célula de uma tabela que contém os dados. Participa no modelo da tabela.

Th

O elemento TH define uma célula de cabeçalho em uma tabela. Os elementos TH estão contidos dentro de um elemento TR (uma linha da tabela), que também pode conter elementos TD para células de dados. Quando o conteúdo de uma célula age como informações de cabeçalho e dados de tabela, TD deve ser usado.

Tr

O elemento TR define uma linha da tabela . Os elementos TR devem estar contidos dentro de um grupo de linhas definido por THEAD , TFOOT ou TBODY
